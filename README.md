# OllamaChatBoot
OllamaChatBoot is a full-stack AI chat application built with Spring Boot, Spring AI, Ollama, and React. The backend connects to a locally running Ollama model using Spring AI, while the frontend provides a simple chat interface for sending prompts and displaying AI-generated responses.

The project demonstrates how to integrate a Java Spring Boot REST API with a React/Vite frontend to build a local AI chatbot application. It is useful for learning Spring AI, experimenting with Ollama models, and understanding how frontend applications can communicate with AI-powered backend services.

## Features

- Spring Boot backend for handling chat requests
- Spring AI integration with Ollama
- REST API endpoint for sending prompts to the AI model
- React/Vite frontend for user interaction
- Local AI model support using Ollama
- Clean project structure with separate backend and frontend folders

## Tech Stack

- Java 17
- Spring Boot
- Spring AI
- Ollama
- React
- Vite
- Maven
- Node.js
